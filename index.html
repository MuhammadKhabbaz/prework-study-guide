HTML

  • The head element contains information about the webpage.
  • The title element is the name of the webpage.
  • The meta element contains metadata about the webpage.
  • The link element links to external resources.
  • The style element contains CSS.
  • The script element contains JavaScript.
  • The base elementdefines the URL
  • The body element represents the visible content shown to the user.
  • The h1,2,3..6 tags represens the level of heading
  • The p tag represents a paragraph.
  • The ul, ol and li tags represent unordered list, ordered list and list items respectively
  • The img tag contains info aboutimages on the webpage, it hasvarious attributes:
  • src: the URL of the image
  • alt: the text to display if the image cannot be loaded
  • width: the width of the image in pixels
  • height: the height of the image in pixels
  • title: the title of the image
  • The br tag creates a line og empty space or a line break between 2 blocks of content
  • The a tag creates a hyperlink to other webpages, it has various attributes:
  • href: the URL of the webpage
  • target: where to open the linked webpage
  • download: whether to download the linked webpage
  • rel: the relationship between the current webpage and the linked webpage

CSS

  • CSS defines the colours, aesthetics, visual layout and organisation of the webpage
  • There are 3 ways to add CSS to a webpage:
  • Inline CSS: directly in the HTML element
  • Internal CSS: in the head element
  • External CSS: in a separate file
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border is a line that goes around the outside of an element.
  • A border-radius rounds the corners of an element.
  • A box-shadow adds a shadow to an element.
  • Selector defines the elements or attributes to with the rules or declerations will apply
  • Declerations contain 2 important elements:
  • Property defines the style of the element
  • Value defines the value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "[decriptive message]": commits your staged content as new commit snapshot
  • git push origin branch-name: pushes the branch to the remote repository
  • git pull origin branch-name: pulls the branch from the remote repository
  • git merge branch-name: merges the branch into the current branch
  • git branch -d branch-name: deletes the branch
  • git log: shows the commit history
  • git diff: shows the difference between the current state and the last commit
  • git reset: resets the current state to the last commit
  • git reset --hard: resets the current state to the last commit and discards all changes

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements allow us to control the flow of our code.
  • Functions are reusable blocks of code that perform a specific task.
  • Arrays are lists of items that can be accessed by their index.
  • Objects are collections of properties that describe a related set of data.
  • Loops allow us to execute a block of code multiple times.
  • Event listeners allow us to listen for events and execute code when they occur.
  • Event handlers allow us to execute code when an event occurs.
  • Event objects contain information about the event that occurred.
